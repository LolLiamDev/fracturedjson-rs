## üìñ Overview

FracturedJson is a tool that formats JSON data to make it easy to read. It organizes information neatly, using smart line breaks and table-like alignment. This version, fracturedjson-rs, is a Rust port of the original FracturedJson project. It keeps your data compact while improving readability, making it ideal for both developers and non-developers alike.

### Why Use FracturedJson?

- **Human-Readable Output**: Get structured and aligned JSON that looks good on the screen.
- **Smart Formatting**: Simple arrays and objects stay on one line, while complex data is formatted elegantly.
- **Optional Comments**: Add notes within your JSON without breaking the format.
- **Cross-Platform**: Install on any operating system that supports Rust.
